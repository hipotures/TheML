# Hypothesis
- title: Hierarchically Shrunk Catalog Template Residuals
- group_name: catalog_template_residuals
- family: template_mismatch
- summary: Build a robust mismatch profile that captures how far each object’s color signature departs from the expected template pattern implied by its catalog tags, so class separation can exploit both conformance and anomaly structure.
- strategy: Compute adjacent colors c1=u-g, c2=g-r, c3=r-i, c4=i-z. For training data, calculate for every (spectral_type, galaxy_population) cell the count n_g, median μ_g,k and MAD s_g,k for each c_k, and the same statistics for each spectral_type alone and for the full training set. For each object, select its cell statistics and define a shrinkage weight w = clip((n_g - 600)/900, 0, 1); if the exact cell is unseen, set w=0 and use the spectral_type profile. Use blended template location and scale: μ*_k = w·μ_g,k + (1-w)·μ_sp,k and s*_k = w·s_g,k + (1-w)·s_sp,k. If the spectral_type group count is also <600, fall back to global μ_0,k and s_0,k. Replace any scale below 0.02 with 0.02 before division. Standardize residuals as z_k=(c_k-μ*_k)/s*_k and clip each z_k to [-10,10]. Emit a compact set of deterministic descriptors: signed z1..z4, mean(|z|), median(|z|), max(|z|), L2 norm of z, and counts of |z_k|>2 and |z_k|>3. This same fallback chain is applied at inference for unseen category combinations.
- expected_signal: Because spectral_type and galaxy_population encode coarse SED families while true class boundaries can still deviate from those templates, these residual and dispersion-based descriptors provide class-specific anomaly signals that can improve recall balance among GALAXY, STAR, and QSO, especially when raw colors overlap.
- risk: If template tags are noisy or drift across train and test, residual centroids may shift and amplify miscalibration; very sparse cells can still produce noisy medians even with shrinkage; and high overlap with raw color features can limit incremental signal while adding preprocessing complexity.
